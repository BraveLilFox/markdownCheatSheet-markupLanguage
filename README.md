## Markdown Cheatsheet

### Basic Syntax
| Element | Markdown Syntax |
| ------ | ----------- |
| Heading | # *H1*<br>## *H2*<br>### *H3*|
|Bold|\*\*bold**|
|Italic|\*italic*|
|Blockquote|> blockquote|
|Ordered List|1. First<br>2. Second<br>3. Third|
|Unordered List|- First<br>- Second<br>- Third|
|Horizontal Rule|---|
|Link|[title]\(www.example.com)|
|Image|![alt text]\(image.jpeg)|
